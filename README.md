# QED プロジェクト：診断モード (SYN) 統合構想 (V1.8)

## 概要
QED ゲームに組み込む隠し診断モード（SYN モード）の設計メモ。
ゲーム共通の保守・デバッグ用コンソールとして、将来的に複数ゲームへ展開する構想。

## UI 仕様
- 起動時に画面を3回白くフラッシュ、全画面黒背景オーバーレイ
- タイプライター風1文字ずつ表示
- レトロ等幅フォント、グリーン (#00FF00) 系
- キーボード入力は `*` でマスク表示、内部で大文字変換

## 認証フロー構造
- Layer 1/2: 一般・上級保守員権限
- Layer 3（隠し階層）: スーパーバイザー権限
- 認証文字列はハッシュ値のみ保持（生文字列をコードに残さない）
- プロンプト文字列は Base64/Hex 難読化

## SubutomoProtocol.js 統合構想
- subutomo-syn/src/SubutomoProtocol.js をゲーム共通ライブラリとして整備
- encode() でスコアコード（QED-XXXX...）を生成
- 複数ゲームから参照できるモジュール構造を目指す
